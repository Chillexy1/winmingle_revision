this project covers the lesson on recursion in c programming language.
    RULES AND REQUIREMENTS.......

    All files are to be compiled on Ubuntu 20.04 LTS using gcc, using the options -Wall -Werror -Wextra -pedantic -std=gnu89.

    All files must end with a newline.

    we're not allowed to use any kind of loop in this project.

    the use of the standard library like printf or puts is FORBIDDEN also.

    also no global variables allowed.

    IMPLEMENTATION OF _putchar is allowed only
