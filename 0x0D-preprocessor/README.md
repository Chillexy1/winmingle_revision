A c preprocessorvis a step performed before a source code is compiled.

rules and requirement needed in this lesson:

    Every file must end with a newline


    No global variables


    No more than 5 functions per file


    Allowed C standard library functions: malloc, free, exit only


    printf, puts, calloc, realloc, etc. are forbidden


    _putchar is allowed: (implement it)


All C files will be compiled on Ubuntu 20.04 LTS using: gcc -Wall -Werror -Wextra -pedantic -std=gnu89
