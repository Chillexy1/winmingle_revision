This is a project that illustrate the use of Static libraries, how it works and how to use one,
also the  use of Unix tools:
    
    ar
    ranlib
    nm

RULES && REQUIREMENTS:

All programs are to be compiled on Ubuntu 20.04 LTS using the Compiler flags: 
    
    -Wall -Werror -Wextra -pedantic -std=gnu89

use of standard libraries like printf, puts, malloc is prohibited.
you can implement a custom putchar.
